# UV-Exposure-Data-Correlation-to-OSHA-Injury-Reporting

## Project Overview
This project analyzes the relationship between UV exposure levels and intensity categories to OSHA injury reporting across various U.S. states. We used publically available datasets to explore whether higher UV intensity in state regions correlates with increased reports of skin-related workplace injuries.

The analysis was conducted using R Studio and Quarto to create an interactive dashboard to help visualize geographic patterns, correlations of UV exposure to region, and  statistical summaries.

## Dashboard
The final interactive dashboard is included in this repository:

- FinalProject_Dashboard.html

Download the HTML file and open it in a web browser to view the interactive dashboard.
